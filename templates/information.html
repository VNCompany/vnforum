Информация о форуме:

Форум предназначен для общения программистов и представляет собой простой аналог киберфорума.

По желанию, его можно опубликовать на публику, при чём конструкция универсальна и подойдёт под любую тематику форума.

Ссылка на github: https://github.com/VNCompany/vnforum

API:

Форум предполагает ещё и свою небольшую систему API.

1. Получение ключа для работы с API:

Для работы с api вы должны быть зарегистрированы в системе и иметь ключ api. Найти его можно у себя в профиле или получить посредством отправки запроса на сервер.

Пример:

            
                curl -X POST http://vncompany.ru/api/token_get -d "{\"login\": \"ваш логин\", \"password\": \"ваш пароль\"}"

Ответ:

            
                {"api_key":"ваш api ключ","status":"ok"}

2. Получение информации о пользователе:

            
                http://vncompany.ru/api/user_get?api=ваш_api&nickname=никнейм_пользователя

3. Получение информации о топике:

            
                http://vncompany.ru/api/topic_get?api=ваш_api&topic_id=id_топика
